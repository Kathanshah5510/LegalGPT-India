# LegalGPT-India

An AI-powered legal assistant built using Retrieval-Augmented Generation (RAG).

## Features (Planned)

- PDF ingestion
- Semantic search using embeddings
- FAISS vector database
- Gemini-powered legal Q&A
- Streamlit web interface
- Source citations
